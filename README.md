## ✏️ 문서 추가/수정 방법

1. 해당 파일 클릭 → ✏️ 아이콘 클릭 → 수정 → **Commit changes**
2. 새 카드 추가: `cards/card-list.md`에 행 추가
3. 새 가이드 추가: `guides/` 폴더에 `.md` 파일 생성
