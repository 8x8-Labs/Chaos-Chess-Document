# 커스텀 기물 가이드

## 개요

이 페이지는 `variants.ini`에 정의된 Fairy-Stockfish 커스텀 기물 목록과 각 기물이 어떤 카드에서 사용되는지를 정리한 레퍼런스 문서입니다.

커스텀 기물은 스킬 카드 효과 적용 시 기존 기물을 일시적으로 교체하는 방식으로 동작합니다. 효과가 끝나면 게임 코드에서 원래 기물로 롤백해야 합니다.

> **주의:** `variants.ini`만으로 표현 불가능한 효과(이동 후 주변 무력화, 반경 접근 제한 등)는 게임 코드에서 별도 처리해야 합니다. 해당 항목은 아래 각 기물 설명에 명시했습니다.
> 

---

## 행마법 기호 범례

| 기호 | 의미 |
| --- | --- |
| `f` | forward (전진 방향) |
| `b` | backward (후진 방향) |
| `W` | Wazir — 직선 1칸 이동 |
| `F` | Ferz — 사선 1칸 이동 |
| `R` | Rook — 직선 슬라이더 (무제한) |
| `B` | Bishop — 사선 슬라이더 (무제한) |
| `Q` | Queen — 직선+사선 슬라이더 |
| `N` | Knight — L자 도약 |
| `K` | King — 전방향 1칸 |
| `n` | nightrider 방향 prefix |
| `p` | 통과 후 잡기 (cannon 계열) |
| `c` | capture only (잡기만 가능) |
| `m` | move only (이동만 가능, 잡기 불가) |
| `숫자` | 최대 이동 칸 수 제한 (예: `4R` = 최대 4칸) |

## 커스텀 기물 목록

### `z` — 나이트라이더

```
customPiece1 = z:nN
```

**사용 카드:** 무한궤도

나이트의 L자 도약 방향으로 계속 직선 이동할 수 있는 기물입니다. 예를 들어 `(0,0)`에서 `(2,1)` 방향으로 이동하면 `(4,2)`, `(6,3)`까지 연속 이동이 가능합니다. 장애물이 있으면 그 앞에서 멈춥니다.

**적용 방식:** 나이트를 이 기물로 교체 → 2턴 후 나이트로 롤백

---

### `y` — 챈슬러

```
customPiece2 = y:RN
```

**사용 카드:** 성 위의 말 (나이트+룩 합체 결과)

룩(직선 슬라이더)과 나이트(L자 도약)의 행마를 모두 가진 기물입니다. 합체 시 나이트가 있던 자리는 비고, 룩 자리에 챈슬러가 생성됩니다.

**적용 방식:** 합체 이벤트 발생 시 룩 위치에 이 기물 생성, 나이트 제거

---

### `x` — 비숍 행마 전용

```
customPiece3 = x:B
```

**사용 카드:** 풍차 (룩 → 비숍 행마로 교체)

사선 방향으로만 슬라이딩 이동하는 기물입니다. 직선 이동은 불가능합니다.

**적용 방식:** 룩을 이 기물로 교체 → 2턴 후 룩으로 롤백

---

### `v` — 룩 행마 전용

```
customPiece4 = v:R
```

**사용 카드:** 풍차 (비숍 → 룩 행마로 교체)

직선 방향으로만 슬라이딩 이동하는 기물입니다. 사선 이동은 불가능합니다.

**적용 방식:** 비숍을 이 기물로 교체 → 2턴 후 비숍으로 롤백

---

### `u` — 앞뒤 이동+잡기 가능 폰

```
customPiece5 = u:fWcfWbWcbWfF
```

**사용 카드:** 민첩

기본 폰에 뒤로 이동(bW)과 뒤로 잡기(cbW) 능력을 추가한 기물입니다.

- `fW` = 앞으로 1칸 이동
- `cfW` = 앞으로 잡기 (기본 폰의 전진 잡기 아님 — 직선 전진으로 잡기)
- `bW` = 뒤로 1칸 이동
- `cbW` = 뒤로 잡기
- `fF` = 전방 사선 이동(기본 폰 잡기)

**적용 방식:** 폰을 이 기물로 교체 → 효과 1회 사용 후 기본 폰으로 롤백

---

### `t` — 전방 사선 이동+잡기 가능 폰

```
customPiece6 = t:fWcfFmfF
```

**사용 카드:** 조준

기본 폰에 전방 사선으로 이동만 하거나 잡기만 할 수 있는 능력을 추가한 기물입니다.

- `fW` = 앞으로 1칸 이동
- `cfF` = 전방 사선으로 잡기만 가능 (이동 불가)
- `mfF` = 전방 사선으로 이동만 가능 (잡기 불가)

**적용 방식:** 폰을 이 기물로 교체 → 1회 사용 후 기본 폰으로 롤백

---

### `s` — 아마존

```
customPiece7 = s:QN
```

**사용 카드:** 정신 집중 (4턴 대기 후 최종 승격 결과)

퀸(직선+사선 슬라이더)과 나이트(L자 도약)의 행마를 모두 가진 최강 기물입니다.

**적용 방식:** 4턴 대기 중 `a`(이동불가 기물) 상태 유지 → 5턴째에 이 기물로 교체

---

### `o`, `i`, `h`, `g`, `j` — 배수진 전진 제한 기물들

```
customPiece9  = o:fR   # 전진 룩
customPiece10 = i:fB   # 전진 비숍
customPiece11 = h:fQ   # 전진 퀸
customPiece12 = g:fN   # 전진 나이트
customPiece16 = j:fK   # 전진 킹
```

**사용 카드:** 배수진

카드 사용 시 자신과 상대 양측의 모든 기물을 각각 대응하는 전진 전용 기물로 교체합니다. 3턴 동안 전진 방향으로만 이동 가능하고, 후진 및 횡이동이 불가능합니다.

| 원본 기물 | 교체 기물 | 허용 행마 |
| --- | --- | --- |
| 룩 | `o` (fR) | 전방 직선 슬라이딩만 |
| 비숍 | `i` (fB) | 전방 사선 슬라이딩만 |
| 퀸 | `h` (fQ) | 전방 직선+사선만 |
| 나이트 | `g` (fN) | 전방 L자 도약만 |
| 킹 | `j` (fK) | 전방 1칸만 |

**적용 방식:** 카드 사용 시 전체 기물 일괄 교체 → 3턴 후 원래 기물로 롤백

---

### `e`, `d`, `c` — 진화(각성) 단계 기물

```
customPiece13 = e:N   # 1단계: 나이트 행마
customPiece14 = d:B   # 2단계: 비숍 행마
customPiece15 = c:Q   # 3단계: 퀸 행마
```

**사용 카드:** 각성 계열 카드

기물이 단계적으로 승격될 때 사용하는 임시 기물입니다. 나이트 → 비숍 → 퀸 순서로 승격됩니다. 실제 기물 종류와 행마법이 동일하지만, 별도 식별자를 사용해 게임 코드에서 승격 상태를 추적합니다.

---

### `a` — 이동 불가 기물

```
customPiece17 = a:
```

**사용 카드:** 성벽, 어둠 속의 손, 무하한, 정신 집중 (대기 단계)

행마법이 없어 어떤 방향으로도 이동할 수 없는 기물입니다. 카드별 세부 용도는 다음과 같습니다.

| 카드 | 용도 |
| --- | --- |
| 성벽 | 빈 칸에 배치하는 장애물. 공격받으면 게임 코드에서 해당 기물 삭제 처리 |
| 어둠 속의 손 | 상대 기물을 이 기물로 교체해 2턴간 이동 불가 구현. 2턴 후 원래 기물으로 롤백 |
| 무하한 | 효과 기물을 이 기물로 교체해 이동 불가 구현. **반경 1칸 접근 제한은 게임 코드에서 별도 처리 필요** |
| 정신 집중 | 4턴 대기 구현. 5턴째에 아마존(`s`)으로 교체 |

---

### `w` — 하극상용 제한 퀸

```
customPiece18 = w:K
```

**사용 카드:** 하극상

퀸을 킹과 동일한 행마(전방향 1칸 이동)로 제한하는 기물입니다. 상대의 퀸을 이 기물로 교체하여 3턴간 강력한 기물을 무력화합니다.

**적용 방식:** 상대 퀸을 이 기물로 교체 → 3턴 후 퀸으로 롤백

---

### `p2` — 신속행군용 2칸 전진 폰

```
customPiece19 = p2:fWfWcfF
```

**사용 카드:** 신속행군

앞으로 2칸 전진이 가능한 폰입니다.

- `fWfW` = 전방으로 1칸 이동 후 추가로 1칸 더 이동 (중간에 장애물 있으면 불가)
- `cfF` = 전방 사선 잡기 (기본 폰 잡기)

> **주의:** 이 기물은 1회성으로 사용 후 반드시 게임 코드에서 기본 폰으로 롤백해야 합니다. 롤백하지 않으면 매 턴 2칸 전진이 유지됩니다.
> 

**적용 방식:** 폰을 이 기물로 교체 → 1턴 후 기본 폰으로 롤백

---

## 게임 코드에서 별도 처리가 필요한 효과

Fairy-Stockfish의 행마법 표기만으로 구현할 수 없는 효과 목록입니다.

| 카드 | 미구현 효과 | 처리 방법 |
| --- | --- | --- |
| 거대화 | 이동 후 반경 1칸 기물 무력화 | 이동 이벤트 후 주변 기물에 `a` 교체 트리거 |
| 무하한 | 반경 1칸 내 접근 시 수 취소 | 이동 검증 단계에서 반경 체크 후 수 거부 |
| 성벽 | 공격 시 벽 파괴 후 해당 칸으로 이동 | 잡기 이벤트 후 `a` 기물 삭제 + 위치 이동 처리 |
| 벽력일섬 | 이동 경로 전체 기물 제거 | 이동 경로 계산 후 경유 칸의 기물 일괄 삭제 처리 |
