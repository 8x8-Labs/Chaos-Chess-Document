# 카드 리스트

> 카드 추가 방법: 아래 테이블에 행을 복사해서 붙여넣고 내용 수정 후 Commit

## 범례

| 항목 | 옵션 |
|------|------|
| **티어** | 일반 / 희귀 / 고급 / 레어 / 전설 / 검토 |
| **타입** | 기물 전용 / 타일 전용 / 전역 |
| **적용 대상** | 폰 / 나이트 / 비숍 / 퀸 / 킹 / 룩 / 모든 기물 / 판 / 타일 / 특정 기물 / 상대 기물 |
| **특수 기믹** | 없음 / 고정 / 나이트라이더 / 챈슬러 / 합체 / 아마존 / 무력화 / 투기장 / 부활 / 벽 / 각성 / 몰락 / 전진 / 교환 / 승격 |
| **개발 현황** | 시작 전 / 진행 중 / 완료 |
| **SFX** | 기본 SFX / 제작 완료 / 제작 중 / 제작 필요 |
