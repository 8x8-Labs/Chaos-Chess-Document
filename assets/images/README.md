# 이미지 폴더

노션에서 사용하던 이미지들을 이곳에 저장하세요.

## 이미지 참조 방법

마크다운 파일에서 아래처럼 상대경로로 사용:

```markdown
![설명](../assets/images/파일명.png)
```

## ⚠️ 노션 이미지 주의사항

노션 S3 이미지 URL은 **1시간 후 만료**됩니다.  
가이드 문서의 이미지를 이 폴더에 직접 저장한 후 경로를 수정해주세요.

필요한 이미지:
- [ ] `example-so.png` — 스킬 카드 가이드 SO 예시
- [ ] `example-code.png` — 스킬 카드 가이드 코드 예시
